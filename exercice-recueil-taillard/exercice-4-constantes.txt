Constantes Indiquer si les constantes ci-après sont correctes ou non. Si non, précisez pourquoi.

1.  1.5         correcte → double
2.  1E3         correcte → double
3.  12u         correcte → unsigned int
4.  12.0u       non correcte, les variables doubles sont toujours signés.
5.  1L          correcte → long
6.  1.0L        correcte → long double
7.  .5          correcte → double
8.  5.          correcte → double
9.  1000000000  correcte → int
10. 0x33        correcte → int
11. 0xefg       non correcte, "g" n'est pas hexadecimal.
12. 0xef        correcte → int
13. 0xEF        correcte → int
14. 0x0.2       non correcte, pas d'écriture hexadecimal pour des doubles.
15. 08          non correcte, 08 n'est pas représentable en octal.
16. 07          correcte → int
17. 0b10        correcte → int
