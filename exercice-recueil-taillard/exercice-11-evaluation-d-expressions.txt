Évaluation d’expressions Donner les valeurs des variables i, j, k après l’exécution de chacune
des expressions ci-dessous.

1. i = (k = 2) + (j = 3);
   i = 5, j = 3, k = 2

2. i = (k = 2) + (j = 2) && (j * 3 + k * 4);
   i = 1, j = 2, k = 2

3. i = (i = 3) || (k = 2) && (j = i + 1) ? (k = j + 2) : (j = k - 1);
   i = 2, j = 0, k = 2
